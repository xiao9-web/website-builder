# 管理后台 UI 重设计规范 v2.0

**项目**：官网构建系统 - 管理后台  
**文档版本**：v2.0  
**日期**：2026-05-12  
**状态**：设计定稿

---

## 一、当前状态分析

### 1.1 现有问题总结

| 维度 | 问题描述 | 严重程度 |
|------|---------|---------|
| 设计一致性 | 颜色、间距硬编码分散各处，无统一设计 token | 高 |
| 侧边栏 | 宽度固定 240px，无折叠动画过渡，无分组 | 高 |
| 登录页 | 渐变背景与后台整体风格割裂，视觉噪声过多 | 中 |
| 仪表盘 | 数据为硬编码 mock，卡片图标使用彩色渐变与整体调性不符 | 中 |
| 表单 | 多处表单样式不统一，无统一 label 宽度 | 中 |
| 暗色模式 | 完全未支持 | 中 |
| 响应式 | 完全硬编码宽度，无移动端适配 | 低（管理端可接受） |
| 空状态 | 各页面无统一空状态处理 | 低 |

### 1.2 技术栈

- Vue 3.4 + TypeScript + Vite
- Element Plus 2.5
- Pinia 状态管理
- CSS scoped 样式

---

## 二、设计目标

1. **统一性**：建立设计 token 体系，所有颜色/间距/圆角引用 CSS 变量
2. **专业感**：深色侧边栏 + 白色内容区，提升管理端专业气质
3. **清晰度**：信息层次清晰，减少视觉噪声，突出关键操作
4. **一致性**：登录页、仪表盘、列表页、编辑页视觉语言统一

### 4.2 侧边栏设计规范

**尺寸**
- 展开宽度：220px（从 240px 缩减，更紧凑）
- 折叠宽度：64px（只显示图标）
- 高度：100vh，fixed

**分区**
- Logo 区：高度 64px，品牌标识
- 菜单区：flex-grow，可滚动
- 用户区：底部固定，头像 + 用户名

**菜单分组标签**
- 浅色小文字分组标题（如 "内容管理"、"网站管理"、"系统"）
- 分组标题 10px，颜色 `--sidebar-text` opacity 0.5
- 折叠时隐藏分组标题，只保留图标

**激活状态**
- 激活项：左侧 3px 蓝色竖条 + 背景 `--sidebar-item-active` 10% 透明度
- 激活文字：白色，图标蓝色

**hover 状态**
- 背景：`--sidebar-item-hover`
- 文字：白色

### 4.3 顶部导航栏

**左侧**
- 折叠按钮（点击展开/折叠侧边栏）
- 面包屑导航（最多2级）

**右侧**
- 用户头像（32px 圆形）+ 用户名 + 下拉箭头
- 点击展开：个人信息 / 退出登录

## 五、登录页设计

### 5.1 设计思路

抛弃紫色渐变背景，改为**几何图形装饰 + 浅灰背景**风格，与管理端深色侧边栏呼应，整体更稳重专业。

### 5.2 视觉结构

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   背景：#F0F4FF（浅蓝灰）                           │
│   左侧：装饰插图区（品牌色几何图形 + 系统说明）      │
│                                                     │
│        ┌─────────────────────┐                      │
│        │ [Logo 图标]          │                      │
│        │ 官网管理系统         │                      │
│        │ ─────────────────── │                      │
│        │ 用户名              │                      │
│        │ 密 码               │                      │
│        │ [登  录]            │                      │
│        │ 默认账号：admin/... │                      │
│        └─────────────────────┘                      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 5.3 规格

- 卡片宽度：420px
- 卡片圆角：`--radius-xl`（12px）
- 卡片阴影：`--shadow-lg`
- 背景：`#EEF2FF`（柔和的蓝紫底）
- 背景装饰：3-4 个不同大小的圆形，颜色 `--color-primary` 10-15% 透明度
- 输入框：高度 42px，边框 `--color-border`
- 登录按钮：高度 44px，背景 `--color-primary`，圆角 `--radius-md`，无渐变

### 6.2 统计卡片规格

**新版设计**（替换彩色渐变）：
- 背景：白色
- 圆角：`--radius-lg`
- 阴影：`--shadow-sm`
- 图标容器：40px × 40px，圆角 `--radius-md`，背景为对应语义色 12% 透明度
- 图标颜色：对应语义色
- 数字：28px，semibold，`--color-text-primary`
- 标签：13px，`--color-text-secondary`
- hover：`--shadow-md`，轻微上移 -2px

**颜色映射**：
- 文章：蓝色 `#4F6EF7`
- 页面：紫色 `#8B5CF6`
- 部署次数：青色 `#06B6D4`
- 最后部署：绿色 `#10B981`

### 6.3 快速操作区

- 4个按钮横向等宽排列（`el-col :span="6"`）
- 每个按钮高度 56px，图标 + 文字竖向排列
- 背景白色，hover 时出现品牌色左侧竖条
- 替代原来的全宽竖向按钮

### 6.4 内容发布横幅

- 保留顶部位置
- 重新设计为左侧文字说明 + 右侧按钮
- 按钮改为非 loading 状态时显示"立即发布"，loading 时显示"发布中..."
- 背景改为品牌色轻底（`--color-primary-bg`），不使用白色卡片

### 8.1 三栏布局

```
┌──────────┬──────────────────────────┬──────────────┐
│ 组件面板  │    画布（预览区）         │  属性面板    │

│ ──────── │  │  拖拽区域          │  │ 属性编辑     │
│ 基础组件 │  │                    │  │ ─────────── │
│ ─ 文本   │  │  [占位符]          │  │ 内容         │
│ ─ 图片   │  │  "拖拽组件到此处"  │  │ [各类编辑器] │
```

### 8.2 工具栏

- 固定在画布区顶部
- 左侧：撤销(Ctrl+Z)、重做(Ctrl+Y)
- 中间：设备预览切换（桌面/平板/手机）
- 右侧：预览、保存、发布
- 高度：48px，背景白色，下边框

### 8.3 组件面板

- 顶部搜索框（过滤组件）
- 分组展示（折叠/展开）
- 每个组件：48px 高，图标 + 名称
- 拖拽时：组件项有蓝色边框提示

### 8.4 属性面板

- 分为 3 个折叠面板：内容、样式、高级
- 默认展开"内容"面板
- 每个字段标签左对齐，输入框右对齐
- 字段间距 `--space-4`

## 九、组件规范库

### 9.1 按钮规范

| 类型 | 使用场景 | Element Plus 配置 |
|------|---------|------------------|
| 主要操作 | 新建、保存、发布 | `type="primary"` |
| 次要操作 | 取消、返回 | `plain` |
| 危险操作 | 删除、清空 | `type="danger"` |
| 文字按钮 | 表格操作列 | `type="primary" link` |
| 图标按钮 | 工具栏操作 | 仅图标，`circle` |

### 9.2 表单规范

- `label-width="80px"` 统一标签宽度
- `label-position="right"` 右对齐
- 验证错误：红色文字，红色边框
- 必填项：`*` 号显示在标签前
- 输入框统一 `size="default"`

### 9.3 卡片规范

```css
/* 标准内容卡片 */
.base-card {
  background: var(--color-bg-card);
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-sm);
  border: 1px solid var(--color-border-light);
  padding: var(--space-6);
}
```

### 9.4 空状态

所有列表页在无数据时展示统一的空状态组件：
- Element Plus `el-empty`
- 图标：相关业务图标
- 文字："{模块名}列表为空，点击 [新建] 开始"
- 操作按钮：与页面新建按钮一致

### 9.5 加载状态

- 表格/列表：使用 `v-loading` 指令
- 按钮操作：`:loading="loading"` 状态
- 页面初始化：骨架屏（使用 `el-skeleton`）
