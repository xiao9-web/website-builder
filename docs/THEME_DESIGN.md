# 主题设计规范

## 一、博客主题（Blog Theme）

### 1.1 主题风格
- **设计风格**：简约、文艺、阅读友好
- **主色调**：#2c3e50（深蓝灰）
- **辅助色**：#3498db（亮蓝）
- **背景色**：#ffffff（白色）
- **文字色**：#333333（深灰）
- **字体**：
  - 标题：思源黑体 / Source Sans Pro
  - 正文：思源宋体 / Georgia

### 1.2 支持的页面类型

#### 1.2.1 首页（Home）
**布局结构：**
```
+----------------------------------+
|           顶部导航栏              |
+----------------------------------+
|                                  |
|         个人简介区域              |
|    （头像 + 昵称 + 一句话介绍）    |
|                                  |
+----------------------------------+
|                                  |
|         文章列表区域              |
|   +------------------------+     |
|   | 文章1                   |     |
|   | 标题 + 摘要 + 日期      |     |
|   +------------------------+     |
|   | 文章2                   |     |
|   +------------------------+     |
|   | 文章3                   |     |
|   +------------------------+     |
|                                  |
+----------------------------------+
|           页脚                   |
+----------------------------------+
```

**配置字段：**
```json
{
  "pageType": "home",
  "profile": {
    "avatar": "头像图片URL",
    "nickname": "博主昵称",
    "bio": "一句话介绍"
  },
  "articles": [
    {
      "id": "文章ID",
      "title": "文章标题",
      "summary": "文章摘要（100字以内）",
      "cover": "封面图URL（可选）",
      "date": "2024-01-01",
      "tags": ["标签1", "标签2"]
    }
  ]
}
```

**样式规范：**
- 个人简介区域：居中对齐，头像圆形 120px，昵称 24px，介绍 16px
- 文章列表：每篇文章卡片，白色背景，hover 时阴影效果
- 文章标题：20px，加粗，深蓝灰色
- 文章摘要：14px，浅灰色，行高 1.6
- 日期和标签：12px，浅灰色

#### 1.2.2 文章详情页（Article）
**布局结构：**
```
+----------------------------------+
|           顶部导航栏              |
+----------------------------------+
|                                  |
|         文章标题                 |
|    （居中，大字号）               |
|                                  |
|    日期 | 分类 | 标签             |
|                                  |
+----------------------------------+
|                                  |
|         文章正文                 |
|    （Markdown 渲染）             |
|                                  |
|    支持：                        |
|    - 标题（h1-h6）               |
|    - 段落                        |
|    - 列表                        |
|    - 代码块                      |
|    - 图片                        |
|    - 引用                        |
|                                  |
+----------------------------------+
|           页脚                   |
+----------------------------------+
```

**配置字段：**
```json
{
  "pageType": "article",
  "title": "文章标题",
  "date": "2024-01-01",
  "category": "分类名称",
  "tags": ["标签1", "标签2"],
  "content": "Markdown 格式的文章内容"
}
```

**样式规范：**
- 文章标题：32px，加粗，居中
- 元信息（日期/分类/标签）：14px，浅灰色，居中
- 正文：16px，行高 1.8，最大宽度 800px，居中
- 代码块：深色背景，等宽字体
- 图片：最大宽度 100%，居中显示

#### 1.2.3 关于页（About）
**布局结构：**
```
+----------------------------------+
|           顶部导航栏              |
+----------------------------------+
|                                  |
|         个人头像                 |
|    （大尺寸，居中）               |
|                                  |
|         个人介绍                 |
|    （多段文字）                   |
|                                  |
|         联系方式                 |
|    （邮箱、GitHub、微信等）       |
|                                  |
+----------------------------------+
|           页脚                   |
+----------------------------------+
```

**配置字段：**
```json
{
  "pageType": "about",
  "avatar": "头像URL",
  "name": "姓名",
  "introduction": "个人介绍（支持多段）",
  "contacts": [
    {
      "type": "email",
      "label": "邮箱",
      "value": "example@email.com",
      "icon": "email-icon"
    },
    {
      "type": "github",
      "label": "GitHub",
      "value": "https://github.com/username",
      "icon": "github-icon"
    }
  ]
}
```

**样式规范：**
- 头像：200px 圆形，居中
- 姓名：28px，加粗，居中
- 介绍：16px，行高 1.8，最大宽度 700px，居中
- 联系方式：图标 + 文字，横向排列，居中

### 1.3 通用组件

#### 顶部导航栏
```
+----------------------------------+
| Logo/站点名    首页 关于 归档    |
+----------------------------------+
```
- 高度：60px
- 背景：白色，底部 1px 边框
- Logo/站点名：左侧，18px，加粗
- 导航链接：右侧，16px，hover 变色

#### 页脚
```
+----------------------------------+
|    © 2024 站点名称. All rights   |
|         reserved.                |
+----------------------------------+
```
- 背景：浅灰色
- 文字：14px，居中，浅灰色
- 高度：80px

## 二、企业主题（Enterprise Theme）

### 2.1 主题风格
- **设计风格**：专业、大气、商务
- **主色调**：#1a73e8（企业蓝）
- **辅助色**：#34a853（绿色）、#fbbc04（橙色）
- **背景色**：#f8f9fa（浅灰）
- **文字色**：#202124（深灰）
- **字体**：
  - 标题：微软雅黑 / Roboto
  - 正文：微软雅黑 / Arial

### 2.2 支持的页面类型

#### 2.2.1 首页（Home）
**布局结构：**
```
+----------------------------------+
|           顶部导航栏              |
+----------------------------------+
|                                  |
|         全屏横幅区域              |
|    （大标题 + 副标题 + 按钮）     |
|         背景图片                 |
|                                  |
+----------------------------------+
|                                  |
|         产品/服务展示             |
|   +--------+  +--------+         |
|   | 产品1  |  | 产品2  |         |
|   +--------+  +--------+         |
|   +--------+  +--------+         |
|   | 产品3  |  | 产品4  |         |
|   +--------+  +--------+         |
|                                  |
+----------------------------------+
|                                  |
|         公司优势区域              |
|    （图标 + 标题 + 描述）         |
|                                  |
+----------------------------------+
|           页脚                   |
+----------------------------------+
```

**配置字段：**
```json
{
  "pageType": "home",
  "banner": {
    "backgroundImage": "背景图URL",
    "title": "企业标语",
    "subtitle": "副标题描述",
    "buttonText": "了解更多",
    "buttonLink": "/about"
  },
  "products": [
    {
      "id": "产品ID",
      "name": "产品名称",
      "description": "产品描述（50字以内）",
      "image": "产品图片URL",
      "link": "/products/1"
    }
  ],
  "advantages": [
    {
      "icon": "图标名称",
      "title": "优势标题",
      "description": "优势描述"
    }
  ]
}
```

**样式规范：**
- 横幅区域：全屏高度（100vh），背景图覆盖，深色遮罩
- 横幅标题：48px，白色，加粗，居中
- 横幅副标题：24px，白色，居中
- 按钮：蓝色背景，白色文字，圆角，hover 变深
- 产品卡片：白色背景，圆角，阴影，4列网格布局
- 产品图片：16:9 比例，顶部
- 产品名称：20px，加粗
- 产品描述：14px，浅灰色

## 三、响应式设计规范

### 3.1 断点定义
- **桌面端**：> 1200px
- **平板端**：768px - 1200px
- **手机端**：< 768px

### 3.2 适配规则

#### 博客主题
- 桌面端：内容最大宽度 1000px，居中
- 平板端：内容宽度 90%，导航栏收起
- 手机端：单列布局，字体缩小

#### 企业主题
- 桌面端：产品 4列，团队 3列
- 平板端：产品 2列，团队 2列
- 手机端：产品 1列，团队 1列

---

## 四、交互规范

### 4.1 动画效果
- 页面切换：淡入淡出，300ms
- 卡片 hover：阴影加深，200ms
- 按钮 hover：背景变深，150ms
- 图片加载：骨架屏 → 淡入

### 4.2 加载状态
- 页面加载：顶部进度条
- 数据加载：骨架屏或 loading 图标
- 图片加载：占位图 → 实际图片

### 4.3 错误处理
- 404 页面：友好提示 + 返回首页按钮
- 加载失败：重试按钮
- 表单验证：实时提示，红色边框

## 五、开发实现要点

### 5.1 主题切换机制
```javascript
// 前端根据当前主题加载对应组件
const themeComponents = {
  blog: {
    home: BlogHome,
    article: BlogArticle,
    about: BlogAbout
  },
  enterprise: {
    home: EnterpriseHome,
    about: EnterpriseAbout,
    product: EnterpriseProduct,
    contact: EnterpriseContact
  }
}
```

### 5.2 配置渲染流程
1. 前端根据路由获取页面配置
2. 根据 pageType 加载对应组件
3. 将 config_json 作为 props 传入组件
4. 组件根据配置渲染内容

### 5.3 样式管理
- 每个主题独立的 CSS 文件
- 使用 CSS 变量定义主题色
- 组件样式使用 scoped

## 六、设计资源

### 6.1 图标库
- 使用 Element Plus 内置图标
- 或使用 Font Awesome

### 6.2 图片规范
- 横幅背景：1920x1080，< 500KB
- 产品图片：800x600，< 200KB
- 头像：200x200，< 50KB
- Logo：支持 SVG 或 PNG

/* 博客主题 */

--blog-secondary: #3498db;
--blog-text: #333333;
