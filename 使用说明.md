# 如何设置首页和菜单

## 1. 创建首页

### 方法一：从模板创建
1. 进入"页面管理"
2. 点击"从模板创建"按钮
3. 选择"首页模板"
4. 系统会自动创建页面并跳转到编辑器
5. 根据需要修改内容
6. 点击"发布"按钮

### 方法二：空白页面
1. 进入"页面管理"
2. 点击"空白页面"按钮
3. 在编辑器中：
   - 点击左侧组件库中的组件添加到页面
   - 选中组件后，在右侧属性面板修改内容
   - 设置组件宽度（100%、50%、33%、25%）实现并排布局
4. 点击"发布"按钮

## 2. 设置为首页

创建页面后，需要在菜单中设置：

1. 进入"菜单管理"
2. 点击"新增菜单"
3. 填写：
   - 菜单名称：首页
   - 关联页面：选择刚才创建的首页
   - 路径：/ （根路径）
4. 点击"保存"
5. 拖动菜单到第一位
6. 点击"保存排序"

## 3. 添加其他菜单

1. 先创建对应的页面（关于我们、产品展示等）
2. 进入"菜单管理"
3. 点击"新增菜单"
4. 填写菜单信息并关联对应页面
5. 拖动调整菜单顺序
6. 点击"保存排序"

## 4. 组件宽度设置（实现并排布局）

选中组件后，在右侧"基础"区域：
- 整行 (100%) - 组件占满一行
- 半行 (50%) - 两个组件并排
- 三分之一 (33%) - 三个组件并排
- 四分之一 (25%) - 四个组件并排

## 5. 文本编辑技巧

文本组件支持HTML格式：
- `<h1>大标题</h1>`
- `<h2>中标题</h2>`
- `<p>段落文字</p>`
- `<strong>加粗</strong>`
- `<em>斜体</em>`
- `<a href="链接">链接文字</a>`

## 常见问题

**Q: 为什么我的首页不显示？**
A: 确保在菜单管理中设置了路径为 `/` 的菜单，并关联了首页。

**Q: 如何让组件并排显示？**
A: 选中组件，在右侧属性面板设置"组件宽度"为50%或33%。

**Q: 修改后前端看不到变化？**
A: 确保点击了"发布"按钮，并刷新前端页面（Ctrl+Shift+R 或 Cmd+Shift+R）。
