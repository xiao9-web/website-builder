# 审计命令集合

## /audit/logs
查看审计日志：
```bash
npm run audit:analyze
```

## /audit/cost
查看成本统计：
```bash
npm run audit:cost
```

## /audit/usage
查看使用统计：
```bash
npm run audit:usage
```

## /audit/performance
查看性能统计：
```bash
npm run audit:performance
```

## /audit/errors
查看错误统计：
```bash
npm run audit:errors
```

## /audit/report
生成完整审计报告：
```bash
npm run audit:report
```
