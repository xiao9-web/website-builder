# 测试命令集合

## /test/unit
运行单元测试：
```bash
npm test
```

## /test/integration
运行集成测试：
```bash
npm run test:integration
```

## /test/e2e
运行E2E测试：
```bash
npm run test:e2e
```

## /test/performance
运行性能测试：
```bash
npm run test:performance
```

## /test/security
运行安全测试：
```bash
npm run test:security
```

## /test/coverage
查看测试覆盖率：
```bash
npm run test:coverage
```

## /test/watch
监听文件变化自动运行测试：
```bash
npm run test:watch
```
